# Text-Summarization Models

This project contains a series of models for **Chinese-Text-Summarization**, and their respective usage. (coming soon!)

- LEAD-3 (Extractive)
- TextRank (Extractive)
- BERTSum (Extractive)
- Pointer-Generation-Network (Abstractive/Generative)
